# MBTI 性格测试系统

一个基于 Web 的 MBTI（迈尔斯-布里格斯类型指标）性格测试系统。

## 功能特点

- 📊 完整的 16 种 MBTI 性格类型测试
- 🎯 精准的四维度评估（E/I, S/N, T/F, J/P）
- 📱 响应式设计，支持移动端和桌面端
- 🎨 美观直观的用户界面
- 📝 详细的性格类型解读

## 使用方法

1. 打开 `index.html` 文件
2. 回答 16 道测试题目
3. 查看您的 MBTI 性格类型结果

## 技术栈

- HTML5
- CSS3
- JavaScript (ES6+)

## 项目结构

```
mbti-test/

├── styles.css          # 样式文件
├── script.js           # 交互逻辑
├── questions.json      # 测试题目数据

```

## MBTI 四个维度

- **E (外向) / I (内向)**: 能量来源
- **S (实感) / N (直觉)**: 信息获取方式
- **T (思考) / F (情感)**: 决策方式
- **J (判断) / P (感知)**: 生活方式

## 16 种性格类型

ISTJ, ISFJ, INFJ, INTJ, ISTP, ISFP, INFP, INTP,
ESTP, ESFP, ENFP, ENTP, ESTJ, ESFJ, ENFJ, ENTJ
